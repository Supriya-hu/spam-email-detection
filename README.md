# 📧 Spam Email Detection using Machine Learning

## 🔍 Project Overview

This project is a Spam Email Detection Web Application built using
Python and Flask. It classifies messages as Spam or Not Spam using a
trained Machine Learning model.

------------------------------------------------------------------------

## 🚀 Technologies Used

-   Python
-   Flask
-   Scikit-learn
-   Pandas
-   HTML
-   Jinja2

------------------------------------------------------------------------

## 🧠 How It Works

1.  User enters a message in the web form.
2.  The message is converted into numerical format using TF-IDF
    Vectorizer.
3.  The trained Naive Bayes model predicts whether the message is Spam
    or Not Spam.
4.  The result is displayed on the webpage.
